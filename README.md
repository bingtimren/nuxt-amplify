## Setup
- Add `nuxt-amplify` dependency using yarn or npm to your project
- Add `nuxt-amplify` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // With options
    ['nuxt-amplify', { /* Amplify configurations */ }],
 ]
}
```
